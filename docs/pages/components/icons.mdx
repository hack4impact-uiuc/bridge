Icons is a wrapper around svg icons we would commonly use in Bridge. You choose the type of Icon to display via the `type` prop. If not given, it defaults to `infoAlert1.

### Alert Icons
These icons may be used in conjuction with [`Alert`](/components/alerts). 

<CodeExampleBox>
<Icon type="errorAlert" />
<Icon type="infoAlert" />
<Icon type="successAlert" />
<Icon type="warningAlert" />
</CodeExampleBox>


```jsx
import { Icon } from '@hack4impact-uiuc/bridge'

<Icon type="errorAlert" />
<Icon type="infoAlert" />
<Icon type="successAlert" />
<Icon type="warningAlert" />
```

### Form Icons
Icons used with forms.
<CodeExampleBox>
    <Icon type="fileDragDrop" />
    <Icon type="opacityUploadFile" />
    <Icon type="uploadFile" />
</CodeExampleBox>

```jsx
import { Icon } from '@hack4impact-uiuc/bridge'

<Icon type="fileDragDrop" />
<Icon type="opacityUploadFile" />
<Icon type="uploadFile" />
```

### Common Props
`Button` includes `COMMON` props. Read [Common Props](/components/common-props) for details and API. These common props will override component props such as the color.

### Component Props

| Prop name | Type | Default | Description |
| :- | :- | :-: | :- |
| type | String | 'infoAlert' | color variant (`errorAlert`, `infoAlert`, `successAlert`, `warningAlert`, `fileDragDrop`, `formErrorAlert`, `opacityUploadFile`, `uploadFile`) |
| theme | Object | Bridge Theme | use to override default bridge theme|
