`Card` is used to contain information together. There are multiple ways to customize the content within the card.

```jsx live
<Card>
    <Card.Content>
        <Card.Title>Card Title</Card.Title>
        <Card.Body>lorem asdfaskdjflkajdslkfj</Card.Body>
    </Card.Content>
    <Card.Footer>
        <Button>"hi"</Button>
    </Card.Footer>
</Card>

```


### Common Props
`Card` includes `COMMON` common props. Read [Common Props](/components/common-props) for details and API. These common props will override component props such as the color.

### Component Props

| Prop name | Type | Default | Description |
| :- | :- | :-: | :- |
| theme | Object | Bridge Theme | use to override default bridge theme|
