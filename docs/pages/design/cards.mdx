#### Card Structure and Spacing

Cards may be any width but whenever used in a group, they should have the same width.

<CardDimensions width="720px"/>

#### Basic Visual Style

**Stationary**

Border Radius: 8 px

Drop Shadow: `box-shadow: 0 2px 5px 0 rgba(0,0,0,0.24);`

<br/>

**Hovered**

Border Radius: 8 px

Drop Shadow: `box-shadow: 0 2px 16px 0 rgba(0,0,0,0.24);`
