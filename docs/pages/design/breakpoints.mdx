**What are breakpoints?**

Breakpoints are points in which, when reached, the content of an interface will adjust its layout to best fit 
the screen size of the user.
It is important for content to be easily digestible and readable for any screen size.

<br />

| Size        | Pixel Value      |
| ----------- |:----------------:|
| X-Small     | 544 px - 767 px  |
| Small       | 768 px - 991 px  |
| Medium      | 992 px - 1279 px |
| Large       | 1280 px +        |
