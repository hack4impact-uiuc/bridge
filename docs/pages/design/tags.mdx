#### Tags Visual Style

**Dimensions**

Tags have horizontal padding of 8 px and vertical padding of 8 px.
Tags have a default height of 28 px.

<br/>

**Borders**

Tags have a border radius of 4 px.

<br/>

**Text**

Buttons use body text style.
