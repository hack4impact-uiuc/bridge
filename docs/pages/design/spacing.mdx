All spacing between text and components are done in increments of 4 pixels.
Typography follows a **baseline grid of 4 px**, so all line heights must be divisible by 4.

| Size        | Pixel Value  |
| ----------- |:------------:| 
| XX- Small   | 4px          |
| X- Small    | 8px          |
| Small       | 16px         |
| Medium      | 32px         |
| Large       | 64px         |
| X- Large    | 128px        |
