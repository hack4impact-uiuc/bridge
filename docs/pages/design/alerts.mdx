### Adding Elements to Alerts

#### Links
Links within alerts are set in 18px HK Grotesk SemiBold text (check [typography](/design/typography)). The link color is the palette's `text` shade. When hovered, the text is underlined.

</Alert>

### Basic Visual Style

**Borders**

Alerts have a border radius of 14 px and a border width of 1px.

<br/>

**Dimensions**

Alerts have vertical padding of 16 px. Width should fill the parent container width.
